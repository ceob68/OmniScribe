# OmniScribe - Guía Rápida

¡Bienvenido a OmniScribe! El centro de control de transcripción en tiempo real.

## 1. Instalación
Ejecuta el archivo `OmniScribe_Setup_v1.0.exe` e instálalo en tu computadora.

## 2. Iniciar la aplicación
Al abrir OmniScribe desde el escritorio, se ejecutará el motor de IA en segundo plano y se abrirá automáticamente tu navegador web con el Dashboard de OmniScribe.

## 3. Empezar a transcribir
1. Asegúrate de tener un micrófono conectado.
2. Haz clic en "Iniciar Grabación" en la interfaz web.
3. Concede el permiso al navegador si te lo solicita.
4. Comienza a hablar y el texto aparecerá mágicamente en la pantalla.
