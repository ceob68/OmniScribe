# Manual de Usuario - OmniScribe
*Versión 1.0*

## 1. ¿Qué es OmniScribe?
OmniScribe es un sistema híbrido que ejecuta inteligencia artificial avanzada en tu propia computadora para procesar voz en tiempo real con latencia casi nula, presentado a través de una interfaz web de lujo.

## 2. Ventajas del Sistema Híbrido
- **Privacidad Total:** Tu voz nunca se envía a internet. Todo se procesa en el servidor Python interno (backend).
- **Flexibilidad Web:** La interfaz Next.js permite un uso increíblemente fluido con animaciones que ningún software de escritorio tradicional puede igualar.

## 3. Uso del Micrófono
OmniScribe requiere acceso a un micrófono. Al ser un entorno web empaquetado (`http://localhost:8000`), los navegadores modernos permiten el uso del micrófono sin restricciones SSL. Si tienes problemas, revisa la configuración de privacidad de Windows.
